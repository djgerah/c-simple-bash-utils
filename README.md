## Описание

Проект состоит из реализации двух стандартных UNIX-утилит — **cat** и **grep** — на языке **C**.  
Обе программы повторяют поведение оригинальных инструментов GNU, поддерживая указанные флаги и комбинации опций.  
Проект направлен на практическое освоение системного программирования, работы с файлами и регулярными выражениями.

## Утилита s21_cat

Утилита **cat** выводит содержимое файлов на стандартный поток вывода.  
Поддерживает следующие флаги:

| №  | Флаг | GNU аналог | Описание |
|----|------|-------------|----------|
| 1  | `-b` | `--number-nonblank` | Нумерует только непустые строки. |
| 2  | `-e` | `-vE` | Отображает символы конца строки как `$`, а также невидимые символы (предполагает `-v`). |
| 3  | `-n` | `--number` | Нумерует все выходные строки. |
| 4  | `-s` | `--squeeze-blank` | Сжимает несколько смежных пустых строк в одну. |
| 5  | `-t` | `-vT` | Отображает табуляции как `^I`, а также невидимые символы (предполагает `-v`). |

## Утилита s21_grep

Утилита **grep** осуществляет поиск строк, соответствующих заданному шаблону (регулярному выражению).  
Реализация использует библиотеку **regex** (или **pcre** при расширенном задании).

### Этап 1 — Базовая версия

Поддерживаются следующие флаги:

| №  | Флаг | Описание |
|----|------|-----------|
| 1  | `-e pattern` | Указывает шаблон для поиска. |
| 2  | `-i` | Игнорирует различия регистра. |
| 3  | `-v` | Инвертирует смысл поиска (показывает несовпадающие строки). |
| 4  | `-c` | Выводит только количество совпадающих строк. |
| 5  | `-l` | Выводит только имена файлов с совпадениями. |
| 6  | `-n` | Добавляет номер строки перед совпадением. |
| 7  | `-h` | Подавляет вывод имени файла перед совпадающими строками. |
| 8  | `-s` | Подавляет сообщения об ошибках для несуществующих или недоступных файлов. |
| 9  | `-f file` | Читает шаблоны из указанного файла. |
| 10 | `-o` | Выводит только совпавшие (непустые) части строк. |

---

Реализуется поддержка **комбинированных опций**, например:

| №  | Флаг | Описание |
|----|------|-----------|
| 1  | `-iv` | Игнорировать регистр и инвертировать поиск |
| 2  | `-in` | Показывать совпадения с номерами строк. |
| 3  | `-cv` | Подсчитать количество несовпадающих строк. |
| 4  | `-niv` | Объединение нескольких опций. |
